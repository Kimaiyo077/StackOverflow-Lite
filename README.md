# StackOverflow-Lite

This is a Q&A web application that allows users to post questions, answer questions and vote for the best question and answer. Users can create accounts to keep track of all their information. The quetions and answers that they post will also be stored along with their information.  

### Prerequisites

A recent brower with javascript enabled to render some animations.

Examples: Chrome(highly recommended), Safari (for mac), Chromium (for linux), Mozilla Firefox.


## Deployment

You can clone the repository to your local machine and run index.html using your favourite browser.
